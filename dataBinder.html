Binder updating a checkbox if the inputs value matches it (0), and Binder having the checkbox update the input.

The use of data-model-force is being used here to force the checkbox to also update the input.

Show Code

<input data-model="check" type="text" value="" placeholder="0 to update checkbox">
<input data-model-force="true" data-model="check" type="checkbox" value="0">

Onclick binding different button shows different text

I show up when the value is 1

I show up when the value is 2

Show Code

<button data-model="onclick" data-click="1">I change model to 1</button>
<button data-model="onclick" data-click="2">I change model to 2</button>

<p data-show="onclick === 1" data-hide="onclick !== 1" style="display:none;">I show up when the value is 1</p>
<p data-show="onclick === 2" data-hide="onclick !== 2" style="display:none;">I show up when the value is 2</p>
