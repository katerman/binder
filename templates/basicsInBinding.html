Basic data binding

using the  data-model=""  attribute on an html element will allow that html element to update the model referenced in the attribute.

Binder will parse all of the  data-model=""  attributes on the page while initiating. However it wont do any filtering such as hiding or showing elements based on model defaults. So you will have to set html hidden/shown yourself.

Attributes

Defaults

Binders default data-attributes can be overwritten incase of naming issues.

  • model (data-model) - used for associating an element to a model  data-model="modelName" 
  • click (data-click) - allows for a element on click to update a model  data-click="modelName = 1" 
  • class (data-class) - Keeps an elements call update to date with a model (can use more than one add a space in between each model)  data-class="modelName1 modelName2" 
  • hide (data-hide) - If the basic expression is true it will hide the element on a domUpdate  data-hide="modelName !== 1" 
  • show (data-show)- If the basic expression is true it will show the element on a domUpdate  data-hide="modelName === 1" 
  • value (data-value) - keeps the associated elements value the same as the model  data-value="modelName" 
  • template (data-template) - updates a dom element with corresponding template
