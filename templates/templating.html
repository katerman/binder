Templating

You can use templating with the data-template attribute. You will have to pass in the id of the script with the same id as your passed in data-template.

The script you use for your template tag must have an invalid type. Suggested to use text/binder-template however this is not crucial. 

<script type="text/binder-template" id="testTemplate">
<p>Gets Replaced by</p>
</script>

<div data-template="testTemplate">
Template Area
</div>
